# java-cli-maven-failsafe-spring-pmd-mockito-serenity-cucumber-junit5-car-data

## Description
A POC for maven app using junit5
mockito, serenity, and cucumber framework
 with pmd
and failsafe plugin.

## Tech stack
- java
- maven
	- spring
  - junit5
  - failsafe
  - pmd
  - cucumber
	- serenity
	- mockito


## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
